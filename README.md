## Installation

  This Calculator system performs Arithmetic Operations.
  
1)Navigate to build
``` 
cd build
```


2)Compile
``` 
cmake ..

```


3)make binaries
``` 
make

```

4)Navigate to calculator_test
``` 
cd calculator_test
```

5)Execute
``` 
cd calculator_tes
```
